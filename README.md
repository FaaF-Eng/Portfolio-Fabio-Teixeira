# 📩 Personal Portfolio Website

I developed a modern and fully responsive personal portfolio website to showcase my professional experience, technical skills, and completed projects in a clean and strategic way. The project features a contemporary design, built with HTML, CSS, and JavaScript to deliver smooth navigation, dynamic interactions, and an optimized experience across all devices.

---

## 📖 About the Project

This project was created to strengthen my digital presence and present my work in a professional and accessible way.

It allows visitors to:

- ✅ Learn more about my professional background and experience
- ✅ Explore technical skills and service areas
- ✅ View selected projects and completed work
- ✅ Navigate through a fully responsive interface
- ✅ Experience smooth animations and dynamic interactions
- ✅ Easily get in touch through contact sections
